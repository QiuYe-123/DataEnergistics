# ME特大インターフェース

<Row gap="20">
<BlockImage id="extendedae:oversize_interface" scale="8"></BlockImage>

ME特大インターフェースは<ItemLink id="extendedae:ex_interface" />と同じスロット数を持ちますが、スロットあたり16倍(Configで変更可能)のアイテムを格納できます。つまり、スロットあたり1024個(64x16)のアイテムを保持できることになります。

他のMODの特定のアイテムには、Configによって特定の乗数を設定できます。
