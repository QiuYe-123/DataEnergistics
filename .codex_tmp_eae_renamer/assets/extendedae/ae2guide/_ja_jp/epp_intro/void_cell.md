# MEボイドセル

ドライブのちっちゃなマターコンデンサー。

<ItemImage id="extendedae:void_cell" scale="4"></ItemImage>

ボイドセルは、使用前に<ItemLink id="ae2:cell_workbench" />で分割する必要があります。フィルターに一致するものはすべて削除されるか、<ItemLink id="ae2:condenser" />のように<ItemLink id="ae2:matter_ball" />または<ItemLink id="ae2:singularity" />などに凝縮されます。

右クリックして設定GUIを開きます。
