# ExtendedAEのはじまり

<Row>
<BlockImage id="extendedae:machine_frame" scale="8"></BlockImage>
</Row>

十分な数の<ItemLink id="extendedae:entro_crystal" />を集めると、最初の拡張型機械フレームをクラフトできるようになります。
これらのフレームはExtendedAEの基礎であり、ExtendedAEのほとんどのブロック型機械のクラフトに必要になります。
