# 設定修飾キット

設定修飾キットは、設定インベントリを一括変更するためのツールです。

<ItemImage id="extendedae:config_modifier" scale="4"></ItemImage>

右クリックしてGUIを開きます。

## 使用方法

設定修飾キットは、インターフェースの設定インベントリなどを、設定によって素早く変更できます。例えば、設定項目の数を最大値に設定したり、すべて削除したりできます。

設定修飾キットを使用して、ワールド内の対象機械(ブロックまたはケーブルパーツ)をクリックすると、その設定を変更できます。

## 設定

**変更モード** と **量変更モード(X)** の2つの主な設定があります。

ボタンをクリックするとモードを変更できます。

![GUI](../pic/cm.png)

- 加算/減算/乗算/除算:設定量の数値をXで加算/減算/乗算/除算します。
- 最大化:設定量を最大値に設定します。
- 最小化:設定量を最小値に設定します。
- セット:設定量の数をXに設定します。
- 消去:すべての設定を消去します。
