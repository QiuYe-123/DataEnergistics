# El comienzo de ExtendedAE

<Row>
<BlockImage id="extendedae:machine_frame" scale="8"></BlockImage>
</Row>

Después de reunir suficientes <ItemLink id="extendedae:entro_crystal" />, finalmente puedes fabricar el primer marco de máquina extendido.
Estos marcos son la base de ExtendedAE, y la mayoría de las máquinas de forma de bloques en ExtendedAE los necesitarán.
