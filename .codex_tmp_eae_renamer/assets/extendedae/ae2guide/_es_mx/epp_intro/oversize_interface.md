# Interfaz ME sobredimensionada

<Row gap="20">
<BlockImage id="extendedae:oversize_interface" scale="8"></BlockImage>
<GameScene zoom="8" background="transparent">
  <ImportStructure src="../structure/cable_oversize_interface.snbt"></ImportStructure>
</GameScene>
</Row>

La interfaz ME sobredimensionada tiene el mismo número de ranuras que la <ItemLink id="extendedae:ex_interface" />, pero puede contener x16 veces (configurable) más
objetos por ranura. Esto significa que puede contener 1024 (64x16) objetos por ranura.

Algunas cosas de otros mods pueden tener un multiplicador específico según la configuración.
